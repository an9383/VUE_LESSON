"추가" 버튼을 만들고 클릭 이벤트를 설정하시오. <button v-on:click='btnAddClick'>추가</button> 3. "추가" 버튼을 클릭하면 행이 추가되게 작성하시오. // 배열명.push( {a:'1', b:'2', c:'3'}) 배열명[배열명.length] = {a:'1', b:'2', c:'3}; 4.
