    설명 npm 명령어 yarn 명령어

    프로젝트 운영 패키지 추가
    dependencies에 추가

    npm install --save 패키지명

    npm install -S 패키지명

    		 yarn add 패키지명
    프로젝트 개발 패키지 추가
    devDependencies에 추가

    npm install --save-dev 패키지명

    npm install -D 패키지명

    		 yarn add --save-dev 패키지명
    전역 패키지 추가

    npm install --global 패키지명

    npm install -g 패키지명

    		 yarn add global 패키지명
