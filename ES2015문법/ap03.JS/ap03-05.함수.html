함수 학습 내용

  1. 함수 정의 방법

    구분 코드
    선언식 함수 정의 
                                        
    var add = function (a, b) {
        return a + b;
    };
    var sum = add(1, 2);
    표현식 함수 정의 
                                        
    function totalsum() {
        var result = 0;
        for (var i = 0; i < arguments.length; i++) {
            result += arguments[i];
        }
        return result;
    }
    var sum = totalsum(1, 2, 3);
    화살표 함수 정의 
                                        
    var add = (a, b) => {
        return a + b;
    }
    var sum = add(1, 3); // 4
    생성자 함수 정의 
                                        
    // 생성자 함수는 객체를 만들 때 사용된다.
    function Goo(name, age) {
        this.name = name;
        this.run = function () {
            return this.name;
        };
    };
    var goo = new Goo('goo', 31);

  2. this의 사용법 학습

    함수 호출 패턴과 this 바인딩

    • 함수에서 this는 window 객체에 바인딩
    • 메서드에서 this는 호출 객체에 바인딩
    • 내부 함수의 this는 window 객체에 바인딩
    • 이벤트 핸들러에서 this는 호출 객체에 바인딩
    • 생성자 함수에서 this는 호출 객체에 바인딩
    • apply에서 this는 호출 객체에 바인딩
