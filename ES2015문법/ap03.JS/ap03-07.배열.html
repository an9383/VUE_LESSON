  1. 배열 선언 방법

    • 배열 리터럴( [] ) 을 이용한 객체 선언
    • Array 생성자 함수를 이용한 객체 선언

  2. 배열 RCUD2S 방법

    생성 let arr = [ ] ;
    추가. Create arr[arr.length] = 값;
    읽기. Read arr[방번호]
    수정. Update arr[방번호] = 값;
    삭제. Delete arr.splice(방번호, 삭제개수)
    검색. Search arr.filter()
    정렬. Sort arr.sort() ✽비교기준: 문자
    복제. arr.map()
    arr.reduce()
    arr.findIndex()

  3. 배열 복사

    • 배열 얕은 복사
    • 배열 깊은 복사

    실행결과예시) [true, 100, "two"]

  3. RCUD2S: Create. 배열 요소 추가

    실행결과예시) [true, 100, "two", "three", undefined, "four", "aaa"]
