npm 사용법 
      React 에서 Webpack 실습을 위해서는 CRA(create-react-app) 를 사용해야 한다.

      # CRA 는 create react app 의 약자.

      # 버전 확인
      $ node -v
      v16.18.1

      $ npm -v
      9.4.2

      $ npx -v
      9.4.2


      # npm 최신 버전 설치
      $ npm install --location=global npm@latest


      $ ls

      $ npx create-react-app webpackprj01
      5분 소요
      pdf 읽기( rt01.docs/데이터변환-복제후할당.pdf )

      # 파일 목록 조회. webpackprj01 폴더가 생성되었는지 확인.
      $ ls

      # webpackprj01 폴더로 이동하기
      $ cd webpackprj01

      # 파일 목록 조회. package.json 파일이 있는지 확인
      $ ls

      # package.json 파일이 있으면 서버 실행하기
      $ npm run start


      # 새로운 터미널을 열고 아래 4개의 파일을 만든다.
      $ ls      # packages.json 파일이 있는지 확인한다.
      node_modules/  package.json  package-lock.json  public/  README.md  src/

      $ touch ./src/App.css
      $ touch ./src/App.js
      $ touch ./src/Header.js
      $ touch ./src/Footer.js
