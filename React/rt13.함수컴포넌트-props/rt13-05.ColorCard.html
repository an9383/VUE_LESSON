CSS로 만든 카드를 컴포넌트로 만들어 본다. 작업 순서 1. 컴포넌트 3개 만들기. CardContainer Card, Square, Label, 2. ReactDOM 을 사용하여 화면에 붙인다. 예시) ReactDOM.render( <CardContainer /> , document.getElementById("app") ) 3. Label 컴포넌트 JSX return ( <div className="label"> {color} </div> ) 4. Square 컴포넌트 JSX let style = { backgroundColor: props.color } return ( <div className="square" style={style}> </div> ) 5. Card 컴포넌트 JSX return ( <div className="card"> <Square color={color}></Square> <Label color={color}></Label> </div> ) 6. CardContainer 컴포넌트 JSX return ( <div> <Card color={"#ff6663"}></Card> <Card color={"#ffa737"}></Card> </div> )
